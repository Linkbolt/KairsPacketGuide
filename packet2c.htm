Ultima Online Protocol :: Death
Packet # Packet Name Length Client Server Obsolete
0x2C Death 0x0002
Data Type Description
byte 0x2C The packet number.
byte Type The type of message being sent.

  • 0x00 - Notify client of his death.
  • 0x01 - Client has chosen to resurrect with penalties.
  • 0x02 - Client has chosen to play as ghost.

Packet Description:
This packet is sent by the server to notify the client of the death of his character.
The client is presented with a menu from which he can choose to resurrect or play as a ghost.
